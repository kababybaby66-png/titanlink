# TitanLink Resources

This folder contains resources bundled with the application.

## Required Files

### ViGEmBus Driver Installer
Download the ViGEmBus driver from:
https://github.com/nefarius/ViGEmBus/releases/latest

Place the installer as:
- `ViGEmBusSetup_x64.msi`

### Application Icon
Create or obtain an application icon:
- `icon.ico` - Windows icon (256x256px recommended)
- `icon.png` - PNG version for other platforms

## Notes

The ViGEmBus driver is required for controller emulation on the host PC.
Without it, hosts won't be able to receive and translate controller inputs
from connected clients.
